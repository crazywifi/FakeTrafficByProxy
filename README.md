# FakeTrafficByProxy
Send Fake Traffic by Free Proxies

## How to use this code?

## Command:
python3 FakeTrafficByProxy.py URL

## Example:
python3 FakeTrafficByProxy.py https://lazyhacker22.blogspot.com/
